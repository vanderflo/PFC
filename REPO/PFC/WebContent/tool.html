      Welcome to UC3M Reporting Tool

       CLOSE

      This Tool has been defined to manage the process of reporting work status in projects with multiple partners assigned. Its friendly interface lets you go through all the aspects of reporting, from work done or results got in a specific task to expenses and effort in a given period of time.
      Take a look on the section below, where basic interactions with the tool are explained.

       Create Project (Administrator)
      •  Click on Projects (header menu)
      •  Click on 'See Projects' (navigation menu, on the left), then ADD NEW PROJECT
      •  Provide basic information (Title, Description, Dates, Coordinator)
       Define Project details (Project Coordinator)
      •  Click on Projects (header menu)
      •  Click on See Projects (navigation menu, on the left), then click on the project you want to edit
      •  Use "Edition Box" section to edit project. These options are available:
        •  Add WP: New Workpackage, with title, description and start/finish dates
        •  Add task: New Task, with title, description and start/finish dates
        •  Assign partner (and effort) to WP: Select partner to be added to this WP and define his effort
        •  Assign partner to Task: Assign partner to a given task
        •  Add reporting deadline date: Define a new reporting deadline
       List Reports and see details
      •  Click on Reports (header menu)
      •  On the Main Menu (left), click on the Project for which you want to list reports
      •  If you are part of this project, you will see the list of subreports for this project
      •  You can filter them by WP or Partner from the menu, or by any character/word of your choice using the search box below the list
      •  To see details of a specific report, click on its date, and all relevant information will be displayed
       Edit a Report
      •  Click on Reports (header menu)
      •  On the Main Menu (left), click on the Project for which you want to list reports
      •  If you are part of this project, you will see the list of subreports for this project
      •  Click on the subreport you want to edit
      •  Use "Edition Box" section to edit report
      •  Edition box:
        •  Change status: Save or Send the report (partner); WP leader will approve/reject it on this section
        •  Add expenses: Specify type of expense, amount and person
        •  Justify effort: Specify effort in person/month
        •  Describe work done for a given task: Provide detils about what has been done and the results of the work
        •  Change flag status: Raise a red flag in case there is any issue blocking the project execution
        •  Add a comment: Use this form to add any comment or feedback you may consider relvant
       Send Report via email
      •  Once you have selected a report to see its details, click on "Send by email" and add an email recipient
